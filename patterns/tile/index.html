Pattern: Tile

A tile is a clickable, single action item. It includes an image, title and description. It's ideally used for displaying news and events.

Example code


<article class="tile">

	<img src='path/to/image.jpg' alt="alt text">

	<h1><a href="#">Title</a></h1>
	
	<p>Description paragraph</p>

	<div class='tile-date' date="2014-08-21">21 August 2014</div>

	<div class='tile-type'>Tile-type</div>

</article>
